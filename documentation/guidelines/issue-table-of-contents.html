Issue table of contents

How to mark up the table of contents for issues

Elements

A table of contents (TOC) can include a range of HTML elements in different combinations. Below is a list of the most used and instructions on how to implement them.

  • Main section heading
  • Section sub-heading
  • Article title
  • Author(s)
  • Page range

Main section heading

Main section headings come after the default heading that always appears, "Contents". To keep the structure of the HTML semantically sound, we use the H3 heading for issue section headings. We use the H3 heading because the page already includes an H1 heading (the issue name) and an H2 heading ("Contents" which introduces the TOC).

Main section heading

<h3 class="serif text-muted">Main section heading</h3>

Sub-headings are available if required.
